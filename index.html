Audio Visual elements

Audio

  • Autoplay: Boolean, use sparingly
  • Controls: provide playback controls for the user
  • Loop: boolean, will play audio over and over
  • Muted: boolean, used to mute the audio on the page
  • src: links to the audio source file
  • Volume: ranges from 0.0 (silent) to 1.0 (loudest)

Video

  • Autoplay: boolean, use sparingly
  • Controls: provide playback controls
  • Height: must be an absolute value, no percentage
  • Muted: boolean, used to mute the page on load
  • Poster: use an alternative image in place of the first frame before playback has initialised
  • src: links to the media source file.
  • Volume: ranges from 0.0 to 1.0
  • Width: can be an absolute value or a percentage
